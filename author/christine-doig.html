The Blaze Ecosystem

Connecting people to data

The Blaze ecosystem is a set of libraries that help users store, describe, query and process data. It is composed of the following projects:

  • Bcolz: A columnar data container that can be compressed
  • Blaze: interface to query data on different storage systems
  • Castra: A partitioned on-disk data structure
  • Dask: Parallel computing through task scheduling and blocked algorithms
  • Datashape: Data description language
  • DyND: A C++ library for dynamic, multidimensional arrays
  • Odo: Data migration between different storage systems
